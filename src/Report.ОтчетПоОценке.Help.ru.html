Отчет по оценкам задач за определенный период

Отчет позволяет проанализировать изменение оценки по задаче за период. 

Особенности использования

  • Оценка 1 показывает сравнение самой первой плановой оценки с фактическим выполнением на сегодняшний день.
  • Колонки анализа некоторой оценки
    • План - плановая оценка установленная в некоторый момент (период);
    • Факт - фактически потраченное время по задаче на дату (период), для "Оценка 1" берется все потраченное время по задаче;
    • Погрешность - считается по формуле: (Факт - План) / План * 100;
    • Период - дата установления уточняющей оценки.
  • Отбор по периоду - отбирает уточнения оценок, которые были установлены в этот период.
