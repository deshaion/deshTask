Справочник календарей

Данный справочник служит для фиксирования выходных дней в календаре.
  • Есть возможность вести несколько календарей, однако использовать в системе пока можно только один. Установить "Календарь по умолчанию" можно в подсистеме "Основная".

Основные действия

  • Редактирование данных производится в пределах года. Изменить текущий редактируемый год можно с помощью спецального поля справа над полем календаря. При этом при смене года будет задаваться вопрос о сохранении измененных данных.
  • Заполнить по умолчанию - в календаре по умолчанию все дни рабочие, кроме субботы и воскресенья
  • Копировать календарь - копирует данные выбранного календаря только за просматриваемый год
