Форма выполненных задач на определенный день

В данной форме отображается хронология работы над задачами за определенный день. Данная форма тесно связана с формой плана задач

Описание элементов главного окна

  1. Группа элементов отображения и установки даты
    Жирным выделена информационная надпись с датой, на которую выведены выполненные дела.
    Есть две кнопки "Вчера" и "Сегодня" для быстрого выбора соответствующей даты. Есть также поле, в которое можно ввести любую дату и увидеть выполненные дела на этот день.
  2. Подменю избранных задач
    Позволяет сохранить некую задачу в избранные, а также быстро начать избранную задачу.
    Для сохранения достаточно выделить строку с сохраняемой задачей и выполнить команду "Избранные\Добавить в избранные".
    Для быстрого начало выполнения избранной задачи, выберете ее в подменю "Избранные".
    Для более подробной информации смотрите в справке к избранным делам
  3. События дня
    Кнопка открывает форму для ввода значимых событий текущего дня (для обсуждения на ежедневном митинге разработчиков).
    Более подробную информацию смотрите в справке к событиям дня.
  4. Группа элементов администритора
    Группа элементов, доступная только пользователям с ролью Полные права. Здесь можно выбрать разработчика и увидеть его выполненные дела на выбранный день.
  5. Таблица выполненных дел на выбранную дату
    Таблица в которой выводится дело, его старт по времени и конец.

Основные особенности использования

Способы добавления задач в таблицу выполненных:

  • Перенести задачу из рядом находящейся формы "План задач"
  • Нажать клавишу Insert, и выбрать необходимую задачу в поле "Дело"
  • Добавить из подменю "Избранные"
Примечание: в системе создана предопределенная задача "Конец рабочего дня". Данная задача служит для фиксирования конца рабочего дня: выбрав эту задачу автоматически фиксируется конец предыдущей задачи на текущее время.

Другие особенности:

  • При изменении старта задачи, одновременно изменяется конец предыдущей задачи. Старт текущей и конец предыдущей задач всегда равны.
  • Есть возможность вводить комментарий по каждой задаче.
